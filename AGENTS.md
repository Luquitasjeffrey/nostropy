# AGENTS Context

Este documento describe el propósito de los agentes (bots, sistemas automatizados) que interactúan o ayudan en el desarrollo de Nostropy. Puede contener instrucciones específicas para asistentes IA o scripts de automatización.

## Roles posibles

- **Desarrollo**: Scripts o asistentes que generan código boilerplate, pruebas unitarias o documentación.
- **Seguridad**: Herramientas que analizan criptografía y sistemas de generación de números aleatorios.
- **Integración con Nostr**: Agentes encargados de publicar y verificar eventos Nostr desde la aplicación.

Los agentes deberían ser configurados para seguir las directrices de seguridad del proyecto y respetar la naturaleza open source. Cualquier información sensible debe manejarse con cautela y no ser incluida en repositorios públicos.
